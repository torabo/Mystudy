# 1. SQL 숫자 연산
```sql
+ 
- 
*
/ : 나누기
DIV : 몫 나누기
% , MOD : 나머지
```

```sql
ABS() : 절대값
CEIL() : 올림
FLOOR() : 내림
ROUND() : 반올림
POW(값,x), POWER() : x거듭제곱
SQRT() : 제곱근
```
